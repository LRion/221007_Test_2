Наиболее важные команды SQL:

SELECT – извлекает данные из базы данных,
ОБНОВЛЕНИЕ – обновляет данные в базе данных,
DELETE – удаляет данные из базы данных,
INSERT INTO (или INSERT) — вставляет новые данные в базу данных,
ORDER BY ASC/DESC – сортирует данные по возрастанию или по убыванию,
CREATE TABLE – создает новую таблицу,
ALTER TABLE – изменяет таблицу,
DROP TABLE – удаляет таблицу,
JOIN — объединяет строки из двух или более таблиц на основе связанного столбца,
FROM — предложение, используемое для перечисления таблиц и любых объединений, необходимых для оператора SQL,
WHERE — предложение, используемое для указания условия при выборке данных из одной таблицы или при объединении с несколькими таблицами.
Команды SQL INSERT, SELECT, UPDATE и DELETE соответствуют четырем основным принципам постоянного хранения: создание, чтение, обновление и удаление (CRUD) соответственно
